## 目前完成功能
```
首页商品的展示 分类商品展示 搜索功能 下拉加载更多商品 商品详情展示 
购物车等操作 收藏商品及查看 
登录账号、注册账号（表单验证）、修改个人信息和头像、头像预览、api响应拦截跳转、操作消息提示
```
